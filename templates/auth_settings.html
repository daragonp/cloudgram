Gestión de Autenticación

Renueva tus tokens de Dropbox y Google Drive directamente desde aquí.

Volver al Dashboard
Dropbox OAuth
Refresh Token Flow

Dropbox requiere un token de acceso "offline" para que el bot pueda conectarse permanentemente. Haz clic abajo para empezar.

Abre la URL de abajo, autoriza la app y pega el código que te den.
Abrir
Copia este valor en tu variable DROPBOX_REFRESH_TOKEN en Railway/Render.
Google Drive API
Credentials JSON Flow

Google Drive utiliza un sistema de JSON completo. Este proceso generará el bloque necesario para tus variables de entorno.

Entra con tu cuenta de Google, permite el acceso y copia el código de respuesta.
Abrir
Este JSON contiene tu Refresh Token. Reemplaza tu variable en la nube.
OneDrive (Azure AD)
MSAL OAuth Flow

OneDrive utiliza la API de Microsoft Graph. Necesitarás autorizar la aplicación para obtener tu Refresh Token.

Inicia sesión en Microsoft, acepta los permisos y **copia el código que aparece en la URL** (parámetro code=...).
Abrir
Copia esto en tu variable ONEDRIVE_REFRESH_TOKEN.
